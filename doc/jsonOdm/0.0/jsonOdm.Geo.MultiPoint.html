MultiPoint

module: jsonOdm

Constructor

new MultiPoint(positions, boundaryBox)

A GeoJSON MultiPoint object

Parameters

Name Type Description
positions Array An array of 2-dimensional arrays with the first entry being the longitude and the second one being the latitude, i.e. [[51,32],[51.4,21]]
boundaryBox Array An array with [min. longitude, min. latitude, max. longitude, max. latitude]

Example

var multiPoint = new jsonOdm.Geo.MultiPoint([
   [51.5,32],[51.6,21]
]);
