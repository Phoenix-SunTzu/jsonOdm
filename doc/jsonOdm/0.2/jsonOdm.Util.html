projectElement(projection, element, parentElement) → {object}

Projects an element to a projectionDefinition

Parameters

Name Type Description
projection object The projection definition object
element object The element that is projected from
parentElement object for internal recursion purpose

Returns

object The projected element

Example

jsonOdm.util.projectElement({
    key1:'value1',
    key2:'value2'
},{
    key1:1,
    key12:function(element){return element.key1 + element.key2}
    // only field modifying queries are allowed
    key2:collection.$query.$branch("key2").$substr(0,3)
})
// will return {key1:'value1',key12:'value1value2',key2:'val'}
